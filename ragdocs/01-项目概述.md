# 基于LangChain4j的RAG知识库系统 — 项目概述

## 1. 项目背景

本项目是一个毕业设计项目，目标是构建一个类似 [Google NotebookLM](https://notebooklm.google.com/) 的 **AI 驱动的知识库管理系统**。用户可以上传自己的文档资料，系统通过 **RAG（检索增强生成）** 技术，让 AI 严格基于用户上传的私有知识进行问答、生成报告和测验。

### 1.1 参考产品：Google NotebookLM

NotebookLM 是 Google 推出的 AI 笔记本工具，核心特点包括：

| 功能 | 说明 |
|------|------|
| **来源管理** | 上传 PDF、文档、网页等作为知识来源 |
| **AI 对话** | 基于文档内容进行 RAG 问答，带引用标注 |
| **报告生成** | 将文档内容转化为结构化报告（简报、学习指南、博客等） |
| **测验生成** | 自动生成选择题测验，支持难度选择、即时反馈、成绩追踪 |
| **Studio** | 统一管理所有生成的产物（报告、测验、笔记等） |

### 1.2 本项目复刻范围

本项目将 **聚焦复刻** NotebookLM 的以下两大核心功能：

1. **📊 报告生成（Report）** — 基于知识库生成结构化文档
2. **📝 测验生成（Quiz）** — 基于知识库生成互动式测验
