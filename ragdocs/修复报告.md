## 三、系统运行前提条件

系统正常运行需要以下条件**同时满足**：

1. **MySQL 已启动**，数据库 `lyhm_ai_rag` 存在，所有表已创建（执行 `sql/user.sql` + `sql/notebook.sql`）
2. **后端已启动**：`./mvnw spring-boot:run -Dspring-boot.run.profiles=local`（端口 8123）
3. **前端已启动**：`cd lyhm-ai-rag-frontend && npm run dev`（端口 5173）
4. **用户已登录**：首次使用需先注册账号，登录后才能访问主页

> ⚠️ 如果后端未启动，前端打开时会弹出"请先登录"并重定向到登录页。这不是 bug，而是认证拦截器的正常行为。
