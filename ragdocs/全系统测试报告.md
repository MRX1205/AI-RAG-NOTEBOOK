## 十一、待注意事项

1. **向量存储复制**：克隆操作复制了向量存储 JSON 文件，使副本可立即使用 RAG 检索。但如果原精选笔记本之后更新了来源，副本不会自动同步。
2. **AI 对话深度测试**：本报告仅验证接口可达性，完整的 AI 问答验证需要有效的 DeepSeek API Key 且服务正常运行。
3. **文件上传测试**：使用测试 PNG 文件验证头像和封面上传，实际用户需上传真实图片。
