# AI-RAG-Notebook：基于 LangChain4j 的 RAG 知识库问答系统

[![Java Version](https://img.shields.io/badge/Java-21-blue.svg)](https://www.oracle.com/java/technologies/downloads/#java21)
[![Spring Boot](https://img.shields.io/badge/Spring%20Boot-3.5.11-brightgreen.svg)](https://spring.io/projects/spring-boot)
[![Vue Version](https://img.shields.io/badge/Vue-3.5.17-brightgreen.svg)](https://vuejs.org/)

## 1. 项目简介
本项目是一个基于 **RAG（检索增强生成）** 技术的 AI 驱动知识库管理系统，旨在复刻 [Google NotebookLM](https://notebooklm.google.com/) 的核心体验。用户可以上传多种格式的私有文档，系统通过向量化存储和智能检索，使 AI 能够严格基于用户提供的知识进行精准问答、结构化报告生成及互动式测验。

## 2. 核心功能
* **📚 来源管理**：支持上传 PDF、DOCX、TXT 等格式文档，利用 Apache Tika 进行高精度解析，并实现文档的向量化存储。
* **💬 RAG 增强对话**：基于文档内容进行 AI 对话，生成的回答支持引用标注，确保信息可追溯。
* **📊 结构化报告**：一键将繁杂的文档内容转化为简报、学习指南或博客文章等结构化文档。
* **📝 互动测验**：自动生成单选/多选测验题，支持难度选择、即时反馈及学习成绩追踪。
* **📂 Studio 工作台**：统一管理所有生成的产物，包括报告、测验和个人笔记。

## 6. 快速开始
### 环境要求
* JDK 21+
* Node.js 18+
* MySQL 8.0+
* DeepSeek API Key (或其它 OpenAi 兼容接口)

### 后端启动
1.  修改 `src/main/resources/application.yml` 中的数据库配置及 API Key。
2.  运行 `AiRagApplication` 类。

### 前端启动
```bash
cd lyhm-ai-rag-frontend
npm install
npm run dev
```
